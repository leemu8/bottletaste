포토샵에서 HTML 문서만들기

Posted by BottleTasteEnt on Aprill 27, 2020

포토샵에서 HTML 문서 만들기 포토샵을 이용하여 모든 레이아웃 작업을 하는 것을 원칙으로 합니다. 초보자들이 비교적 안전하게 화면설계기획서에서 제시한 내용대로 정확하게 레이아웃 배치 작업할 수 있다는 장점과 드림위버에서 작업 시 화면 정열이 깨지는 문제를 미리 방지할 수 있다는 장점이 있습니다. 따라서 포토샵에서 완성된 홈페이지 이미지를 드림위버로 넘어가기 이전에 화면 그대로를 HTML 문서로 자동 저장해주는 포토샵의 슬라이스 기능은 꼭 알아야할 필수 기능입니다.

우선 포토샵에서 웹사이트의 레이아웃을 구성합니다. [도구상자]의 [돋보기 도구]를 선택하고 CTRL+R을 눌러 작업 창의 줄자옵션을 활성화합니다. 이미지 아래에 가이드라인을 정확히 설정해줍니다. 가이드 라인과 슬라이스 간의 스냅 기능이 활성화되므로 자석처럼 붙는 느낌으로 작업합니다. 분할영역 작업이 마무리되면 SHIFT+CTRL+S를 눌러 PSD 파일로 저장합니다.

포토샵에서 바로 HTML 문서로 저장하기 위해 메뉴바에서 웹용으로 저장 메뉴를 클릭합니다. [웹용으로 저장] 팝업창이 나타나면 화면 왼쪽의 [도구박스]에서 [분할 영역 선택 도구]를 클릭하고 화면의 아무 부분이나 한 번 클릭해줍니다. 포맷을 JPG로 설정하고 [저장] 버튼을 눌러줍니다.

이상으로 웹디자인에 대해서 알아보았습니다. 저는 건축 전공이지만 웹디자인에도 관심이 있습니다. 건축도 도면 구성하는 부분에 있어서 레이아웃 구성도 중요합니다. 마찬가지로 웹디자인 또한 가독성이 편하고 추구하고자 하는 부분은 강조하고 조절하는 레이아웃 구성은 비슷한 거 같습니다. 프로그램 공부도 하면서 디자인도 어느 정도 잘한다면 금상첨화가 아닐까? 싶습니다.

[최적화 다른 이름으로 저장] 팝업창이 나타나면 파일 이름은 원하는 대로 입력하고 반드시 파일 형식을 "HTML 및 이미지로 선택합니다. 이런 식으로 진행하여야 HTML 파일을 포토샵에서 만들 수 있습니다. HTML 파일과 이미지 파일의 폴더가 생성되는 것을 알 수 있습니다.

INDEX.HTML 파일을 더블클릭하여 웹브라우저에서 열어보면 완성된 웹페이지가 보입니다. 드림위버로 작업 시 사용할 수 있는 기초 소스가 됩니다. 테이블로 어렵게 레이아웃을 만든다고 고생할 필요가 없어지게 됩니다.

이상으로 포토샵으로 HTML 문서만드는 과정을 설명해 보았습니다. 웹페이지를 만들기 위해서는 기본 레이아웃을 잡아야 하는데 이를 도와줄 수 있는 프로그램이 포토샵입니다. 포토샵을 통해 레이아웃 구성을 쉽게 구성하여 코드를 작성하면 더 쉽게 웹사이트를 제작할 수 있게 됩니다. 저도 포토샵을 통하여 다양한 웹디자인을 해볼 생각입니다.
